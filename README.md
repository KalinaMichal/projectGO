### Usuwanie starej i instalowanie najnowszej wersji Java w systemie LINUX Ubuntu

$ sudo update-alternatives --remove "java"
$ cd /usr/lib/jvm
$ sudo rm -r java-1.8.0-open-amd64
$ sudo rm -r java....
$ sudo rm -r jdk-13.0.1
$ sudo rm -r jvm
$ cd $HOME
$ nano .bashrc file
